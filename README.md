# 📄 Chat with PDF using RAG (Retrieval-Augmented Generation)
#### This notebook demonstrates how to build a simple chatbot that answers questions from a PDF document using RAG (Retrieval-Augmented Generation).
## It leverages:

- 🦜 LangChain for chaining and document processing

- 🦙 Ollama for local LLM inference (like LLaMA2, Mistral, etc.)

- 📚 Chroma as a vector store for document embeddings

- 🧠 Unstructured for parsing PDFs

- 🤖 RAG pipeline for contextual responses

## 🧩 Features
1. Upload and process any PDF file.

2. Chunk and embed the document.

3. Store embeddings in ChromaDB.

4. Ask questions, and get context-aware answers from your local LLM using RAG.
